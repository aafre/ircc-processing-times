# IRCC Visa Processing Times Tracker

Live processing times dashboard for Canadian visa applications, powered by official IRCC data.

## Features

- Real-time processing times for 212+ countries
- Filter by visa type: Visitor, Super Visa, Study, Work
- Alphabetical range navigation (A-E, F-J, K-O, P-T, U-Z)
- Sort by fastest or slowest processing times
- Auto-updates daily from IRCC data
- Built for r/CanadaVisitorVisa

## How it works

The app fetches official IRCC processing time data daily and displays it as an interactive dashboard post in your subreddit. Moderators can force-refresh data or create new dashboard posts via the subreddit overflow menu.

## Installation

1. Install the app on your subreddit
2. Use the `...` menu > "Create Processing Times Dashboard" to create a dashboard post
3. Use "Refresh Processing Times Data" to fetch the latest data

Data is sourced from Immigration, Refugees and Citizenship Canada (IRCC).
